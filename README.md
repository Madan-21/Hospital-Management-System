# Hospital Management System

A Python command-line application for managing hospital operations including doctors, patients, admissions, discharges, and administrative tasks.

## About

This Hospital Management System is a Python application designed to streamline hospital operations through a command-line interface. It provides comprehensive functionality for managing doctors, patients, admissions, discharges, and various hospital administrative tasks.

## Installation

1. Clone the repository:
```bash
   git clone https://github.com/Madan-21/Hospital-Management-System.git
```

2. Navigate to the project directory:
```bash
   cd Hospital-Management-System
```

3. Run the application:
```bash
   python Main.py
```

## Usage

Launch the application by running `Main.py`. The command-line interface will guide you through various options:

- Managing doctor records
- Adding and updating patient information
- Processing admissions and discharges
- Accessing administrative functions
- Viewing and managing person records

## Modules

### Admin.py
Handles administrative functions including system configuration, user management, and oversight operations.

### Doctor.py
Manages all doctor-related operations such as adding new doctors, updating information, and viewing doctor schedules.

### Patient.py
Handles patient records, medical history, and patient-related operations.

### Person.py
Base module for managing person entities in the system, providing common functionality for users, doctors, and patients.

### Main.py
The entry point of the application that initializes the system and provides the main menu interface.
